Expense Tracker
===============

**Expense Tracker** is a simple yet powerful tool for managing and analyzing your personal expenses.
It allows users to record transactions, categorize expenses, and generate insightful reports.

**Features:**
- Track daily expenses
- View summaries by category or time period
- Store data locally in SQLite
- Command-line interface for quick usage
